# Eval Scorecard

> Generated by `pnpm eval`. Measures how well the scoring rubric agrees with hand-labeled
> expected tiers and ordering.

## Labeled dataset (`packages/eval/src/labels.json`)

~10–12 of the 25 leads hand-ranked into expected `tier` and relative `expectedOrder` (1 = best),
each with a one-line rationale. Anchored to the stratification in [PLAN.md §1](../PLAN.md).

## Metrics

| Metric | Description |
|---|---|
| **Tier accuracy** | Fraction of labeled leads where predicted tier matches expected tier. |
| **Confusion matrix** | 5×5 (hot / strong / lukewarm / weak / spam); rows = expected, cols = predicted. |
| **Spearman ρ** | Rank correlation between expected ordering and predicted `qualityScore`. |
